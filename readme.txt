== Description ==

<strong>Polar Facebook Auto Poster</strong> is a FREE WordPress plugin for auto post to your facebook account. You can use this to publish your site's posts,pages and custom post types to your facebook profile and fan pages as soon as you hit the publish button in your site.
The posting process is fully automated and as easy as 1, 2, 3!

Auto Publish WordPress posts/pages/custom post types to your facebook profile and fan pages.

* <strong>Configure your facebook account</strong> on which you want to auto post.
* <strong>Define your content </strong>– configure which content you’d like to auto post (eg. posts,pages and custom post types, select post format, message format and category filters )
* <strong>Relax </strong>– your every new content will be automatically posted on your facebook increasing the audence and making them viral.<br />
Update your fanbase about your content, wide spread your content and make your content viral!

= Features: =

* <strong>Auto post on your facebook profile and fan pages</strong>
* <strong>Auto post of new posts, custom post types (you can configure)</strong>
* <strong>Auto Post filtering as per post types  and taxonomies i.e you can enable auto post for certain post types or certain categories</strong>
* <strong>Logs available to track all your the auto posts to individual accounts</strong>
* <strong>Post format selection ability</strong> - You can post either simple text message or share your blog post.
* <strong>Message format configuration ability</strong> -
* <strong>Ability to publish a single wordpress post to facebook profile and fan pages concurrently with just one publish click</strong>
* <strong>Ability to enable and disable auto post in post screen</strong> - you won't need to go to plugin settings to disable Auto Post
* <strong>Lightweight</strong> - Installs in seconds
* <strong>User friendly</strong> - very easy to configure and setup with all the necesssary step guidance on plugin settings page.

== Installation ==
1. Unzip polar.zip
1. Upload all the files to the /wp-content/plugins/polar
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. For customizing the plugin's settings, click on Social Auto Poster menu in Wordpress left admin menu.


== Frequently Asked Questions ==
= What does this plugin do? =
This plugin provides the ability to to post your wordpress posts/pages/custom post types to your facebook fan page and profile page automatically.

= How will auto post work? =
At first, you will need to get all the required credentials and setup the plugin in the plugin settings page and as soon as you configure the plugin completely, then whenever you will publish the post then it will be automatically posted to your facebook fan pages and profile page as per your configuration.

= Where can I find the documentation for the plugin? =
Once you install the plugin , you can check some general documentation about how to use the plugin in the "How to use" panel of the plugin's settings page.
